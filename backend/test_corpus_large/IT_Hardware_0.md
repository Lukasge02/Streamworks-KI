
    # IT Standard: Hardware
    Status: Active
    Gültig ab: 2025-01-01
    
    ## 1. Zweck
    Diese Richtlinie regelt den Umgang mit Hardware im Unternehmen.
    
    ## 2. Geltungsbereich
    Gilt für alle Mitarbeiter der Streamworks GmbH.
    
    ## 3. Details
    Der Standard-Timeout für Hardware-Verbindungen beträgt 30s. Version 1.0.
    
    ## 4. Ausnahmen
    Ausnahmen bedürfen der schriftlichen Genehmigung durch CTO.
